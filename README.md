# 橙影 · Codex 企业级工作流 Skill Pack

这是一个用于 Codex 长期项目开发的技能包。

目标：让 Codex 不直接乱写代码，而是按以下流程工作：

PM访谈 → 视觉确认 → 开源扫描 → 技术架构 → 安全评估 → 项目分级 → 任务拆解 → 开发执行 → 文档沉淀 → 回滚治理

## 快速使用

把本目录中的文件复制到你的项目根目录，然后对 Codex 说：

```text
启动橙影 Codex 企业级工作流，先 PM 访谈，不要直接开发。
```

## 适用项目

- 简单网站
- 小程序
- App
- SaaS 项目
- 后台管理系统
- 企业内部系统
- 工具软件
- 长期商业项目
- 基于开源项目二开

## 建议

先跑通 Skill，不要急着做控制台系统。确认 Codex 能稳定按流程提问、总结、等待确认后，再考虑自动化、多线程、架构图和控制台。
