# REVIEW_CHECKLIST.md

## PR / 分支审查清单

### 范围

- [ ] 是否只完成当前任务？
- [ ] 是否修改无关模块？
- [ ] 是否引入新依赖？
- [ ] 是否有顺手重构？

### 质量

- [ ] TypeScript 是否通过？
- [ ] Lint 是否通过？
- [ ] Build 是否通过？
- [ ] 是否有重复代码？
- [ ] 是否有无用 import？

### 安全

- [ ] 是否暴露 API Key？
- [ ] 是否影响权限？
- [ ] 是否影响上传？
- [ ] 是否影响用户数据？

### 数据

- [ ] 是否修改数据结构？
- [ ] 是否有 migration？
- [ ] 是否向后兼容？
- [ ] 是否可回滚？

### 文档

- [ ] CHANGELOG 是否更新？
- [ ] TASKS 是否更新？
- [ ] DECISIONS 是否需要更新？
- [ ] CONTEXT 是否需要更新？
